üìñ About Live2D Integration

This demo showcases the PixiLive2DRenderer integration for Deep Tree Echo. It uses the pixi-live2d-display library to render Live2D Cubism models in the browser.

To use your own models, you can:

  • Download sample models from Live2D Sample Data
  • Create your own models with Cubism Editor
  • Host model files on a CDN or your own server
  • Reference models by URL or local path
